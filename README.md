# Hero Women Driver

A Next.js web application for Hero Women Driver, a trusted taxi service connecting students with reliable women drivers for safe and secure transportation.

## Features

- Student-Focused: Designed specifically for students and parents who prioritize safety
- Women Drivers: Connects users with verified female taxi drivers
- Modern UI: Built with Next.js, TypeScript, and Tailwind CSS
- Responsive Design: Optimized for all devices
- Contact Forms: Easy booking and inquiry system

## Tech Stack

- Framework: Next.js 15
- Language: TypeScript
- Styling: Tailwind CSS
- UI Components: Radix UI
- Icons: Lucide React
- Forms: React Hook Form with Zod validation

## Getting Started

1. Clone the repository:
git clone https://github.com/hannahabebe/hero-women-driver.git
cd hero-women-driver

2. Install dependencies:
npm install

3. Run the development server:
npm run dev

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- app/ - Next.js app router pages
- components/ - Reusable React components
- lib/ - Utility functions
- public/ - Static assets (images, icons)
